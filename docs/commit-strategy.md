# Commit Strategy

1. `chore: initialize project structure and .gitignore`
2. `feat(backend): initialize express backend with modular architecture`
3. `feat(frontend): initialize react vite app with routing and theme`
4. `database: add supabase schema, rls policies, indexes, and views`
5. `feat(auth): implement signup/login/me/logout/password-reset`
6. `feat(subscriptions): add stripe checkout and webhook handling`
7. `feat(scores): add rolling five-score management`
8. `feat(charities): add admin charity management and user charity selection`
9. `feat(draws): add simulation/publish/jackpot rollover services`
10. `feat(winners): add winner verification and payout workflow`
11. `test: add backend service and route tests`
12. `chore: deployment config and environment hardening`
